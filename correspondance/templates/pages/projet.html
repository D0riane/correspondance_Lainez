Le projet


Ce projet a été mené dans le cadre de l'évaluation finale du cours de Python de M. Clérice, à l'École nationale des chartes, pour l'année académique 2020-2021.

Les consignes étaient de "développer une application avec base de données relationnelle, comprenant formulaire pour ajout, suppression, édition. Il doit être possible de naviguer dans la collection, d'y faire une recherche simple voire complexe, un index peut être ajouté."

Correspondance Jésuite est une application web pensée comme un outil collaboratif pour centraliser l'importante correspondance reçue par le général Lainez, Préposé général de la Compagnie de Jésus entre 1558 et 1565. Pour l'instant, elle rassemble les lettres issues de quelques volumes numérisés de la Monumenta Historica Societatis Iesu, concernant principalement la correspondance européenne.

Elle répond à deux objectifs :

  • Consulter les données : Les données sont consultables par tous. Chacun peut naviguer pour consulter les lettres, les transcriptions et les sources répertoriées dans la base de données.
  • Participer à l'enrichissement des données en ajoutant, éditant et supprimant du contenu, c'est-à-dire des lettres, des transcriptions et des sources. Ces fonctionnalités sont réservées aux utilisateurs. Après s'être inscrit, l'utilisateur peut participer à ce projet collaboratif.
Toutes les données sont ouvertes et peuvent être récupérées en JSON.

Ce projet a été réalisé par Doriane Hare.
