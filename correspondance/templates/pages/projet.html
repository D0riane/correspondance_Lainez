Le projet Correspondance Jésuite


Cadre du projet

Ce projet a été mené dans le cadre de l'évaluation finale du cours de Python de M. Clérice, à l'École nationale des chartes, pour l'année académique 2020-2021.
Il a été réalisé par Doriane Hare.

Consignes

Les consignes étaient de développer une application web avec base de données relationnelle, comprenant formulaire pour ajout, suppression, édition. Il devait être possible de naviguer dans la collection, d'y faire une recherche simple voire complexe, un index pouvait être ajouté.

L'application

Correspondance Jésuite est une application web pensée comme un outil collaboratif pour centraliser l'importante correspondance reçue par le général Lainez, Préposé général de la Compagnie de Jésus entre 1558 et 1565. Elle rassemble les données issues de quelques volumes numérisés de la Monumenta Historica Societatis Iesu, concernant principalement la correspondance européenne.

Les objectifs

  • Consulter les données : Les données sont consultables par tous. Chacun peut naviguer pour consulter les lettres, les transcriptions et les sources répertoriées dans la base de données.
  • Participer à l'enrichissement des données en ajoutant, éditant et supprimant du contenu, c'est-à-dire des lettres, des transcriptions et des sources. Ces fonctionnalités sont réservées aux utilisateurs. Après s'être inscrit, l'utilisateur peut participer à ce projet collaboratif.
  • Toutes les données sont ouvertes et peuvent être affichées en JSON en cliquant sur le bouton suivant le standard JSONAPI 1.0.

Aspects techniques
  • Cette application web a été développée avec le microframework Flask.
  • La base de données relationnelle a été développée avec l'outil DB Browser for SQLite.
  • L'ORM SqlAlchemy a été utilisé pour dialoguer avec le SGBD.
